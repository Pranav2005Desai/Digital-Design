# Digital Design

This repository contains Verilog/SystemVerilog projects and practice modules for digital design and RTL development.

The goal of this repository is to build strong fundamentals in:
- RTL design
- combinational and sequential logic
- digital architecture concepts
- simulation and waveform debugging
- clean hardware design workflow

---

# Repository Structure

## Practice Modules

- `muxes/`  
  Multiplexer designs and variations

- `adders/`  
  Adder architectures and arithmetic circuits

- `converters/`  
  Binary, Gray code, and Excess-3 converters

Additional modules and projects will be added progressively.
