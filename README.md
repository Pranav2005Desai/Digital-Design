# Digital Design

This repository contains Verilog/SystemVerilog modules for digital design practice.

## 📂 Structure
- **converters/** — Binary/Gray/Excess-3 converters
  - `src/` → source code (RTL)
  - `tb/` → testbenches
  - `cons/` → constraint files
- **scripts/** — TCL or automation scripts
- **docs/** — notes or block diagrams

Each converter is an independent Vivado project with its own HDL, testbench, and constraints.
